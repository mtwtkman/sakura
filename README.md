# What
This is a wsgi based toy webframework influenced by the interface of [actix-web](https://github.com/actix/actix-web).
No dependencies, not for production.

# Required
- python3.7+
- poetry

# Development
```sh
$ make init
```
Just write code.

# Test
```sh
$ make t

```

# Example
```sh
$ make example
```
